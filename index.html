ABOUT US

Latch Network is a security-first blockchain of C-Lite Wallets.

Latch Network functions as an all-encompassing layer of access control for all of Web3, bringing real-world use cases in a completely decentralised manner.

Introducing C-Lite Wallets

A C-Lite Wallet is decentralized, programmable, and transferable, and can sign transactions on any other blockchain.

Do you believe that the C-Lite Wallet will improve usability for end users?

The C-Lite Wallet will be a revolutionary feature, and all tokens using it will be able to enhance their own user experience tremendously by removing blocking times and making transactions much easier.

02
How would you use a C-Lite Wallet marketplace?

The C-Lite Wallet is designed to provide a set of unified and frictionless APIs that empower developers to build applications that must or can tolerate multiple blockchain and token interfaces - it was not designed to be an application itself, but it would be possible for an application built with the C-Lite API to be an integrated wallet marketplace.

03
How does the MPC work in a client-server context?

All the systems we've implemented for C-Lite are also part of the Smart Contract Rollout Spec and Part D of our Full Node Spec, so that anyone can replicate the results in their own system. In other words, there's a wealth of documentation and software available to verify our claims if we haven't provided enough details ourselves.

04
What are your biggest fears regarding the future of the Latch Network?

The Latch Network is particularly concerned with security and the security principle in general. We have designed and implemented a user design that ensures that all C-Lite Wallets are more than capable of signing transactions. The security of the network will be further enhanced by using TLS encryption, which prevents eavesdropping of the data transiting from the user's browser to their C-Lite Wallet.
